# Complex Analysis Review

A multi-page GitHub Pages site for reviewing complex analysis / 复变函数.

## Local preview

Open `index.html` directly in a browser.

## Structure

- `index.html` — overview
- `basics.html` — basic definitions
- `analytic.html` — analyticity and Cauchy-Riemann equations
- `integral.html` — contour integrals and Cauchy theorem
- `formula.html` — Cauchy integral formula and consequences
- `series.html` — Taylor/Laurent series, singularities, residues
- `exam.html` — exam patterns
- `flashcards.html` — active recall cards
- `checklist.html` — final review checklist
